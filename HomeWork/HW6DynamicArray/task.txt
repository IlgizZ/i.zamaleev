Динамический массив (dynamicArray)
Прочитать про списки. (Например, Ахо, Ульман “Алгоритмы и структуры данных”)

Реализовать следующую структуру данных: безразмерный массив, способный хранить в себе любые объекты. Массив изначально имеет некоторую вместимость. По умолчанию 10. При попытке положить туда 11-ый элемент, массив перестраивается, его длина увеличивается до 15. Если длина списка была равна 20 и из него убрали 6 элементов, то его длина становится равной 15. То есть длина массива увеличивается на 5 каждый раз, когда это необходимо. И уменьшается на 5, когда удалено больше 5 элементов. 
Требования к реализации: создать класс DynamicArray, содержащий методы
void addElement(Object) - добавляет элемент в список
void insertElement(Object) - добавляет элемент в список с нужным индексом
void removeElement(Object) - удаляет элемент из списка
Object getElement(int) - возвращает элемент с данным индексом
int getSize() - возвращает размер списка
