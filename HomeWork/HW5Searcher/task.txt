Поиск файлов (searcher)
Необходимо написать консольную программу для поиска файлов с данным наименованием. Пример работы программы: java Main [опции] строка1 строка2.
Опции:
-R (-r) - рекурсивный поиск по вложенным папкам
-D (-d) - указать до какой вложенности следует искать файл. Формат определения опции: -d n, где n - натуральное число
-r и -d не могут быть указаны вместе. По умолчанию поиск производится в текущей папке
-v (-V) - нужно ли включать в результаты поиска папки с подходящим названием. По умолчанию включать их не нужно
строка1 - папка, в которой нужно произвести поиск
строка2 - часть наименования файлов. Например, поиск по строке “as” должен дать результат: ask_me.txt, castle.jpg, past.doc.
Программа должна ругаться и правильно падать при неправильном вводе. Как пример, можете смотреть существующие консольные утилиты. (Можете посмотреть в сторону org.apache.commons.cli)
